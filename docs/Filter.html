filterProducts function

For all filters (color, brand, main category), it adds or removes "hidden" class according to info in globalFilter object
If no filter is selected, it shows all products.
After filters are looped, it updates filter count (function). Finally, it checks whether there are any products left. If not, it shows "no products found" message.
